 📌 Overview

This project is an interactive Netflix Analytics Dashboard built using Power BI. It provides insights into Netflix's content library through dynamic KPIs, interactive visualizations, and geographic analysis. The dashboard helps users explore trends in content distribution, genres, ratings, release years, and global availability.


 🚀 Dashboard Preview

[Netflix Dashboard](https://github.com/Programmer-anamika/Netflix-Dashboard/blob/main/Netflix_Dashboard.png)

 ✨ Features

- 📊 Interactive KPI Cards
- 🎭 Genre-wise Content Analysis
- ⭐ Content Rating Distribution
- 🌍 Country-wise Content Map
- 🍿 Movies vs TV Shows Comparison
- 📅 Release Year Analysis
- 🎯 Dynamic Filters and Slicers
- 📈 Clean and Interactive Dashboard Design


 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Figma (Dashboard Background Design)

 📈 Key Insights

- Analyze the distribution of Movies and TV Shows.
- Discover the most popular content genres.
- Explore Netflix content across different countries.
- Understand content ratings and release year trends.
- View key metrics through dynamic KPI cards.
